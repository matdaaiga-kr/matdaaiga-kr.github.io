2026년 03월 28일, 2026년 Global AI Agent Camp가 경북대학교에서 열렸습니다. 

Global AI Agent Camp는 전세계적인 커뮤니티인 Global AI Community에서 매년 주최하는 행사로, 대구 챕터에서는 Microsoft Agent Framework(MAF)를 활용한 지능형 앱을 제작해보는 실전 워크샵을 준비했습니다.

![행사 전체 사진][image-00]

하나의 앱에서 여러 AI 에이전트가 협업하는 모습을 직접 경험해볼 수 있었고, AI 기술과 도구의 가능성을 발견할 수 있는 소중한 시간이었습니다.

### Session0. 개발 환경 설정 | 유저스틴

![session0][image-01]

Microsoft에서 Principal Developer Advocate으로 재직 중인 유저스틴님께서는 Microsoft Agent Framework의 전반적인 소개와 개발환경 설정을 주제로 세션을 준비하셨습니다.

Microsoft Agent Framework는 Microsoft에서 제공하는 AI 에이전트 개발 프레임워크로, 단일 에이전트부터 다중 에이전트까지 다양한 시나리오에 적용할 수 있는 유연한 솔루션입니다.

- [유저스틴님의 다양한 개발 소식 링크드인에서 받아보기][justin-sns]

### Session1. Microsoft Agent Framework 사용해서 단일 에이전트 개발하기 | 황지현

![session1][image-02]

[Microsoft 학생 홍보대사][msa]로 활동 중인 황지현님께서는 Microsoft Agent Framework로 단일 에이전트를 개발하는 실습 세션을 준비하셨습니다.

Microsoft Agent Framework로 직접 LLM을 연결하여 쉽게 AI 에이전트를 개발해보는 경험을 얻을 수 있었습니다.

- [황지현님의 다양한 개발 소식 링크드인에서 받아보기][jihyeon-sns]

### Session2. Microsoft Agent Framework에 프론트엔드 UI 연동하기 | 황지현

![session2][image-03]

[Microsoft 학생 홍보대사][msa]로 활동 중인 황지현님께서 Microsoft Agent Framework에 AG-UI를 연결하여 프론트엔드와 연동하는 실습 세션을 준비하셨습니다.

처음부터 프론트엔드 앱을 개발해야 하는 것이 아닌 Microsoft Agent Framework로 에이전트를 개발한 뒤, AG-UI를 활용하여 프론트엔드와 쉽게 연동하는 모습이 인상적이었습니다. 

- [황지현님의 다양한 개발 소식 링크드인에서 받아보기][jihyeon-sns]

### Session3. Microsoft Agent Framework 사용해서 다중 에이전트 개발하기 | 이시영

![session3][image-04]

코멘토에서 Education Manager로 근무중인 이시영님께서는 Microsoft Agent Framework로 다중 에이전트를 개발하는 실습 세션을 준비하셨습니다.

세션1에서 단일 에이전트를 개발해보았다면, 이번 세션에서는 다중 에이전트를 개발하여 여러 AI 에이전트가 협업하는 모습을 경험해볼 수 있었습니다.

- [이시영님이 알려주는 다양한 개발 소식 링크드인에서 받아보기][siyoung-sns]

### Session4. Aspire로 프론트엔드 웹 UI와 백엔드 에이전트 오케스트레이션하기 | 이시영

![session4][image-05]

이시영님께서 Aspire로 프론트엔드 웹 UI와 백엔드 에이전트 오케스트레이션하는 실습 세션을 준비하셨습니다.

Aspire는 Microsoft에서 제공하는 에이전트 오케스트레이션 플랫폼으로, 프론트엔드 웹 UI와 백엔드 에이전트를 쉽게 연결하여 효율적으로 관리할 수 있는 솔루션입니다.

대시보드를 통해 에이전트의 상태를 모니터링하고, 프론트엔드와 백엔드 간의 원활한 통신을 경험할 수 있었습니다.

- [이시영님이 알려주는 다양한 개발 소식 링크드인에서 받아보기][siyoung-sns]

### Session5. MCP 서버 개발하기 | 김근희

![session5][image-06]

Microsoft에서 Education Success Manager로 재직 중인 김근희님께서는 MCP 서버 개발에 대해 발표해주셨습니다.

MCP 서버는 에이전트가 외부 시스템과 통신할 수 있도록 하는 서버로, Microsoft Agent Framework에 사용하면 에이전트에 여러 도구를 추가할 수 있습니다. 

김근희님께서는 MCP 서버의 기본적인 개념과 개발 방법에 대해 설명해주셨습니다.

- [김근희님이 알려주는 다양한 개발 소식 링크드인에서 받아보기][geunhee-sns]

### Session6. Microsoft Agent Framework에 MCP 서버 연동하기 | 윤금재

![session6][image-07]

Microsoft에서 Education Success Manager로 재직 중인 윤금재님께서는 Microsoft Agent Framework에 MCP 서버를 연동하는 방법에 대해 발표해주셨습니다.

MCP 서버를 연동함으로써 에이전트가 외부 시스템과 원활하게 통신할 수 있으며, 다양한 도구와 기능을 에이전트에 추가할 수 있는 방법을 배울 수 있었습니다.

- [윤금재님이 알려주는 다양한 개발 소식 링크드인에서 받아보기][keumjae-sns]
---

이번 행사를 제대로 운영할 수 있도록 지원해주신 Microsoft, 경북대학교 IT대학 컴퓨터학부, (주)해달프로그래밍 관계자 분들께 다시 한 번 감사의 말씀을 전합니다.

앞으로도 맞다AI가에서는 대구/경북 지역의 개발자들이 빠르게 최신의 AI 기술과 클라우드 네이티브 기술을 접할 수 있도록 다양한 이벤트를 준비할 예정입니다. 다가올 4월 밋업도 많은 기대 부탁합니다!
