전 세계 수천 명의 개발자가 같은 날, 같은 주제로 모이는 글로벌 커뮤니티 이벤트! Global Azure Bootcamp가 올해는 대구에서 열립니다.

Global Azure Bootcamp은 매년 전 세계 각 도시에서 동시에 개최되는 Microsoft Azure 클라우드 기술 행사입니다. 클라우드 아키텍처, DevOps, 확장 가능한 솔루션 등 Azure의 다양한 기술을 함께 배우고 나눌 수 있는 자리입니다. 학생, 개발자, 클라우드 기술에 관심 있는 누구나 환영합니다!


👉 [지금 참가하세요!][event-register]

## 행사 개요

- 일시: 2026년 4월 18일(토), 12:00~18:00
- 장소: 대구 스케일업 허브 (DASH) - 5층 대회의실
- 참가대상: 학생, 개발자, 클라우드 기술에 관심 있는 누구나
- 세션 주제: 추후 공개 예정

## 키노트 스피커 소개

이번 Global Azure Bootcamp 2026 in Daegu의 키노트 연사로, Microsoft Principal Developer Advocate인 **Henk Boelman**이 네덜란드에서 대구까지 직접 찾아옵니다! 🎉 

AI, Machine Learning, Azure, DevOps 분야의 전문가이자 Microsoft 입사 전 Microsoft AI MVP로 활동했던 Henk Boelman은, 많은 Azure 기반 AI 플랫폼 구축 경험을 바탕으로 개발자들이 머신러닝에 쉽게 입문할 수 있도록 전 세계를 누비며 지식을 나누고 있는 분입니다. 

그런 분이 대구에 온다고요? 네, 진짜입니다. 글로벌 무대에서 활약하는 Microsoft Principal Developer Advocate의 키노트를 대구에서 직접 들을 수 있는 이 기회, 절대 놓치지 마세요!
