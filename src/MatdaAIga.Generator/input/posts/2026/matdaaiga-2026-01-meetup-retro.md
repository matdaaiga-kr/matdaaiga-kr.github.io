2026년 01월 28일, 2026년 맞다AI가의 첫번째 밋업, "Hello, Cloud/AI!"이 대구 스케일업 허브(DASH)에서 열렸습니다.

맞다AI가는 대구/경북 AI 개발자 커뮤니티로, 2024년 12월부터 활동하고 있습니다. AI에 관심있는 누구나 참여할 수 있고, 다양한 최신 AI 기술을 발빠르게 전달하고 있습니다.

![행사 전체 사진][image-00]

어제 열린 01월 밋업에서 다룬 내용은 개발자의 클라우드/AI 첫걸음이었는데요. 

직접 부딪히며 배운 Azure 클라우드, AI 입문기를 공유하는 자리였습니다. 

현직자에게는 신선한 자극을, 학생들에게는 "나도 할 수 있다"는 자신감을 채울 수 있는 소중한 시간이었습니다. 

### Session1. Colab을 넘어, Azure와 PowerBI로 완성하는 데이터 시각화 | 신정빈

![session1][image-01]

신정빈님께서는 Azure와 PowerBI로 데이터 시각화를 하는 데모를 보여주셨습니다. 

Colab에서는 데이터를 정적인 그래프로만 확인할 수 있었는데요. 이것을 넘어 Azure와 PowerBI를 활용해 실시간, 동적 시각화를 하는 모습이 인상깊었습니다. 

[신정빈님의 다양한 개발 소식 링크드인에서 받아보기][jungbin-sns]

### Session2. Azure 기반의 이미지 분석 이해하기 | 여지빈

![session2][image-02]

여지빈님께서는 Azure에 기반한 이미지 분석에 대해 말씀해주셨습니다. 

어렵고 복잡하게만 느껴졌던 이미지 분석을 Azure를 활용해서 쉽게 할 수 있는 모습을 재미있게 보여주셨습니다. 

[여지빈님과 링크드인에서 소통하기][jibin-sns]

### Session3. Microsoft Agent Framework 초간단 소개 | 유저스틴

![session3][image-03]

Microsoft에서 Principal Developer Advocate으로 재직 중인 유저스틴님께서는 Microsoft Agent Framework에 대해 아주 간단히 소개해주셨습니다. 

작년 밋업의 주제가 되었던 Semantic Kernel의 확장 버전인데요, 앞으로 Agent Framework가 어떻게 활용될 수 있을지 가능성을 엿볼 수 있었습니다. 

[유저스틴님이 알려주는 다양한 개발 소식 링크드인에서 받아보기][justin-sns]

앞으로도 맞다AI가에서는 대구/경북 지역의 개발자들이 빠르게 최신의 AI 기술과 클라우드 네이티브 기술을 접할 수 있도록 다양한 이벤트를 준비할 예정입니다. 다가올 2월 밋업도 많은 기대 부탁합니다!
